## Security Risk Assessment

The overall risk of the repo is low. Though it does contain accadmic work, there is no true security risks of personal data such as card information or important features of an individual. 

- **Academic integrity:** If the repository were made public or accessed by other students, assignment solutions could be copied, and accademic misconduct could be completed. 

- **Personal data:** If any datasets used in assignments contain real personal information, unauthorized access could raise privacy concerns. Any such data should be anonymized or excluded from version control via `.gitignore`.
